Telemetry

This provides several screens to display telemetry when using a long range telemetry module.

Note that the telemetry can also talk, you can choose the voice and event that you want to say inthe application configuration

The following screens are available:

- A graph showing the current altitude and events information

- Information about the altimeter such as altimeter type/firmware, output continuity, current altitude etc..

- For the alti GPS this provide an aditionnal screen to display phone and rocket location informations using their GPS

- For the alti GPS this provide a map which shows the phone and rocket positions on the map. This will show you where the rocket is going.

Note that to track your rocket you should use the track screen which will memorize the last known rocket position
