Choix du module bluetooth

Vous serez ammené à choisir un module bluetooth si aucun n'a été sélectionné ou si vous voulez vous connecter à un altimètre différent.

Vous devez d'abord appairer tous les modules bluetooth de vos altimètres dans la configuration de votre appareil Android. Faites attention à bien les différencier de manière à vos connecter au bon altimètre

A partir de la liste des modules configuré sélectionnez celui utilisé par votre altimètre, quitter l'écran et connectez vous à nouveau. Si vous avez choisi le mauvais module la connexion ne s'établira pas.
