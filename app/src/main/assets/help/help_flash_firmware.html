Flash firmware

First you need to select the firmware. The firmware depend on the altimeter that you want to flash

After using a recover firmware you will need to flash the firmware.

In order to be able to flash your altimeter you will need a USB programmer and an adapter to plug it to your device.

a USB adapter (USBC or micro USB depending on your Android device)

Currently only the chair pcb is supported

AltiMultiESP32

Flashing Chair PCB

- Plug the cable

- Select the correct firmware using ... Button

Then press the ... button. The firmware will now be uploaded, if not try again.

IMPORTANT NOTE: when flashing, do not disconnect the cable, this will corrupt the firmware and you will have to flash it again
