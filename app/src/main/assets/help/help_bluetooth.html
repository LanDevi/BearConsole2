Select bluetooth module

You will be prompted to choose a bluetooth module type if you have not selected one already or if you want to connect to a different altimeter.

You first need to pair all you bluetooth altimeter modules in your Android device bluetooth configuration. Make sure that you can clearly differenciate them so that you connect to the correct altimeter

From the list of configured modules select the one your altimeter is using, leave the screen and connect again. If you have choosen the wrong module it will not connect.
