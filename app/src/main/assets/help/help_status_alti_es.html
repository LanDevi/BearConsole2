Altimeter status

This provides several screens to check the altimeter status.

- Information about the altimeter such as altimeter type/firmware, output continuity, current altitude etc..

- Gives the ability to test the pyro output or servos. Use it with caution and make sure that no pyro charges are connected while testing

- For the alti GPS this provide an aditionnal screen to display phone and rocket location informations using their GPS

- For the alti GPS this provide a map which shows the phone and rocket positions on the map
