File di aiuto per la schermata di volo

Il volo è diviso in 2 schermate:

- Vista delle curve che traccia tutte le curve di volo. In questa schermata puoi scegliere quali curve visualizzare.

- Informazioni sul volo. In questa schermata avrai un riepilogo del volo. È inoltre possibile esportare tutte le curve in formato CSV.
