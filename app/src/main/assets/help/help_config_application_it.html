Configura l'applicazione Bear Console

La configurazione è divisa in 2 schermate

Generale

- Lingua dell'applicazione (non utilizzata per ora perché la lingua dell'applicazione sarà la lingua del telefono)

- Colore del grafico

- Unità dell'applicazione

- Colore di sfondo del grafico (assicurati che sia diverso dal colore del grafico !!!)

- Dimensione del carattere

- Baud rate (questo è per il cavo USB, deve essere lo stesso del baud rate dell'altimetro)

- Tipo di connessione può essere Bluetooth o USB se vuoi fare telemetria a lunga distanza)

- Libreria grafica (consente di scegliere tra 2 tipi di lib, fai attenzione che su Android 8 e successivi farai crashare il tuo telefono se non utilizzi la libreria MPAndroid chart)

- Multiplo Main / drogue Ti consente di definire più di un'uscita come principale o una drogue

- Supporto USB completo Abilitalo solo se usi USB con un cavo. Se si utilizza la telemetria a lunga portata, non utilizzarlo per evitare di corrompere le configurazioni dell'altimetro

Configurazione della telemetria

Consente di configurare quali eventi di telemetria vengono detti. Consente inoltre di modificare la voce della telemetria. Se vuoi provare la voce della telemetria, premi semplicemente il pulsante [Test voce]

Quando hai finito, fai clic sul pulsante Salva. È inoltre possibile uscire dalla configurazione senza salvare o ripristinarla alla predefinita.
